# Plainsound Text
A somewhat condensed, Univers-inspired sans serif with integrated Helmholtz-Ellis Just Intonation accidentals.

Currently in development.

## Typing accidentals
| Accidental description | Unicode | Type this |
|------------------------|---------|-----------|
| Flat                   | E260    | *f        |
| Natural                | E261    | *n        |
| Sharp                  | E262    | *s        |
| Double sharp           | E263    | *S        |
| Double flat            | E264    | *F        |
